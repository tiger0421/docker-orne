# Usage

```
curl https://raw.githubusercontent.com/tiger0421/docker-orne/master/launch.sh > launch.sh
sudo chmod a+x launch.sh
sh launch.sh
```

```
docker build -t IMAGE_NAME:TAG_NAME .
```
